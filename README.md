# To Run
1.  Open command line cmd at the root of the repository.
2.  Run the command `python Tool_Word_Analytics.py --end_point [Enter HTML End Point Here]` OR `python Tool_Word_Analytics.py --ep [Enter HTML End Point Here]`.
3. For further help enter the command `python Tool_Word_Analytics.py --h`.
4. The Summary statistics appears as output at the terminal.
